Dub Gap Finder

I created an AudioSuite plugin for ProTools that analyzes a reference track, and then renders a source track to detect missed/low volume dialogue to help quicken the QC process. This was written in C++ using ProTools' AAX development suite. It works for Dolby 5.1, Stereo 2.0 and Regular Mono audio.

Mifare Hacking

While working at Allthenticate, I took frequent trips to conferences like DEFCON and HOU.SEC.CON as a volunteer at the Physical Security Village. Here I demonstrated methods on how to exploit vulnerabilites in weakly encrypted cards like the Mifare classics, MagStripes and regular HID cards. My GitHub has a simple Dockerfile that can be run in conjecture with cheap hardware that can be used to dump/clone/corrupt card data.

Borland Theme for Intelli-J

When I first started to learn how to code, it was an incredibly old-school compiler called Turbo-C on a shady Windows XP virtual machine. While Turbo-C has been extinct since the Ice Ages, I decided to bring that nostalgic blue look to JetBrains' Intelli-J IDE (something I used wayyyy too much during my Postgrd program).
